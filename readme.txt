== Description ==
This plugin is used to create the carousel that can be inserted to any page. This plugin support wordpress multisite setup.

== Support ==
Visit the [plugin's home page](http://wordpress.org/extend/plugins/wpnewcarousels/) to leave comments, ask questions, etc.

== Installation ==
1. Unzip
2. Upload to your plugin directory
3. Enable the plugin

== Using the WPNewCarousels ==

1. Add New carousel in settings page 
2. Add data to new carousel carousel in main 
3. Use shortcode [wpnewcarousel name="YOUR_CAROUSEL_NAME" height="" width=""] <br />
4. Height and Width are the optional parameters if inserted then these will replace the default Height and Width
5. name is required parameter
